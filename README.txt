Nationwide Logistics Inc - Truck and Trailer Repair Work Orders

Open the tool:
Double-click "Open Garage Desk.bat" or open "index.html" in your browser.

What it does:
- Create and edit truck and trailer repair work orders
- Create a work order by pasting an email request
- Save the repair email ID customers should send requests to
- Read "Reported Problem" from customer emails
- Use voice input for reported problems, labor, parts, completion notes, and pasted email text
- Track Pending, In Progress, Awaiting Parts, Awaiting Approval, Ready for Review, Completed, and Closed / Invoiced jobs
- Set and filter work order priority: Normal, Urgent, Roadside, or Fleet
- Track parts needed and parts costs
- Track technician, priority, labor hours, labor cost, and completion notes
- View a dashboard, work order list, detailed work order page, and vehicle list
- View detailed work orders with labor and parts line items
- Track vehicles separately from work orders
- Update truck/trailer files with make, model, year, engine, DOT date, registration date, last service, next PM due, and notes
- Bulk update truck/trailer files by importing a CSV file from your computer or pasting CSV-style rows
- Branded Nationwide Logistics Inc logo and browser tab icon

Project structure:
- entities/: Vehicle and WorkOrder models
- pages/: Dashboard, WorkOrders, WorkOrderDetail, Vehicles
- components/: Layout, StatsCard, WorkOrderCard, VehicleForm, WorkOrderForm, StatusBadge

Your work orders are saved in this browser on this computer.
